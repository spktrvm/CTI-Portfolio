Welcome to my Cyber Threat Intelligence (CTI) and Malware Analysis repository.

## 📂 Repository Structure

* **[`/Reports`](./Reports/)**: Tactical and strategic threat analysis reports. Includes deep dives into malware campaigns, infection chains, and actionable mitigation strategies.
* **[`/Scripts_and_Tools`](./Scripts_and_Tools/)**: Python automation scripts designed to accelerate the intelligence collection phase, including automated IOC enrichment leveraging external APIs.
* **[`/YARA_Rules`](./YARA_Rules/)**: Custom YARA rules written to detect specific malware families, offensive tools, and campaign-specific artifacts based on host and network indicators.
